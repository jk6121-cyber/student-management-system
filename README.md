#  Student Management System

##  Project Description
A React-based Student Management System that allows users to perform CRUD operations like adding, editing, deleting, and searching student records. It also uses Local Storage to store data permanently in the browser.

---

##  Features
- Add Student Details
- Edit Student Information
- Delete Student Records
- Search Students
- Local Storage Data Persistence

---

##  Tech Stack
- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- Axios
- Local Storage

---
